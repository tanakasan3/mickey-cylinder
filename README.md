# Mickey Cylinder

OpenSCAD model for a small cylinder with Mickey Mouse-style ears.

## Specifications

- **Body:** 3mm diameter Ã— 8mm height
- **Center hole:** 1mm diameter (through entire height)
- **Ears:** Two round cylinders on opposite sides at the top

## Usage

Open `mickey_cylinder.scad` in OpenSCAD and render (F6) to generate the model. Export as STL for 3D printing.

## Customization

Edit the parameters at the top of the file:
- `body_diameter` - Main cylinder diameter (default: 3mm)
- `body_height` - Main cylinder height (default: 8mm)
- `hole_diameter` - Center hole diameter (default: 1mm)
- `ear_diameter` - Ear size (default: 1.5mm)
- `ear_height` - Ear thickness (default: 2mm)
- `ear_offset` - Ear position from center (default: 0.8mm)
